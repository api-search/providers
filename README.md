# API Search - Providers

This repository stores provider data for the [APIs.io](https://apis.io) search network. Each provider represents a company or organization that publishes APIs indexed by the network.

## About

Provider records are generated from [api-evangelist](https://github.com/api-evangelist) repos and used to build the `_providers/` Jekyll collection on the APIs.io network site.

## Related Repositories

- [apis](https://github.com/api-search/apis) - Individual API records
- [tags](https://github.com/api-search/tags) - API tags and categories
- [capabilities](https://github.com/api-search/capabilities) - Workflow and capability definitions
- [schemas](https://github.com/api-search/schemas) - JSON Schema references

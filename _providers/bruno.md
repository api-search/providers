It stores collections directly on the filesystem as plain text files using the Bru markup language, making collections version-controllable alongside code. Bruno supports REST, GraphQL, and gRPC APIs, scripting with JavaScript, environment variables, pre-request and post-response hooks, and CI/CD automation via Bruno CLI.

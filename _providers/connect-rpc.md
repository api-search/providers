description: Connect is a slim framework for building browser-friendly and gRPC-compatible HTTP APIs from Protocol Buffer schemas. Servers built with Connect speak the Connect protocol, gRPC, and gRPC-Web simultaneously, while clients can switch between protocols without changing code. Connect provides type-safe SDKs for Go, TypeScript/JavaScript, Swift, Kotlin, and Python, plus the Buf-generated `protoc-gen-connect-*` code generators. The project is open source under the Apache 2.0 license and is maintained by Buf.
